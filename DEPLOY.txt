========================================
公网部署手把手（让别人也能用 Kokoro）
========================================

目标：
  别人用任意浏览器打开你的网址
  → 就能文本生成语音 + Rhubarb 嘴型预览

推荐配置：
  - 云服务器：2核 4GB 内存起（建议 4核 8GB 更稳）
  - 系统：Ubuntu 22.04
  - 开放端口：3921（网页）
  - 硬盘：至少 20GB（Kokoro 模型约几百 MB + Docker 镜像）

----------------------------------------
第 1 步：买一台云服务器
----------------------------------------
任选一家（国内常用）：
  - 腾讯云轻量：https://cloud.tencent.com/product/lighthouse
  - 阿里云轻量：https://www.aliyun.com/product/swas

购买时选择：
  1. 地域：离你近的（如上海 / 广州）
  2. 镜像：Ubuntu 22.04
  3. 套餐：尽量选 4GB 内存以上
  4. 记下：
     - 公网 IP（例如 1.2.3.4）
     - 登录密码 或 SSH 密钥

买好后，在控制台「防火墙 / 安全组」放行：
  - TCP 22（SSH，用来登录）
  - TCP 3921（网页服务）

完成第 1 步后回复我：
  「服务器买好了，系统是 Ubuntu，公网 IP 是 xxx」
我会继续带你第 2 步。

----------------------------------------
第 2 步：用 SSH 登录服务器（预告）
----------------------------------------
Windows 可用：
  - PowerShell
  - 或下载 FinalShell / PuTTY

命令示例：
  ssh root@你的公网IP

----------------------------------------
第 3 步：安装 Docker（预告）
----------------------------------------
在服务器执行官方一键脚本，然后用 docker compose 启动本项目。

----------------------------------------
第 4 步：上传项目并启动（预告）
----------------------------------------
把本项目传到服务器后执行：
  docker compose up -d --build

首次启动会下载 Kokoro 模型，可能要几分钟。

----------------------------------------
第 5 步：验证
----------------------------------------
浏览器打开：
  http://你的公网IP:3921

应看到：
  - 语音代理已连接
  - Kokoro 已就绪
  - Rhubarb 引擎已就绪

然后把这个链接发给别人即可试用。

----------------------------------------
安全提醒（很重要）
----------------------------------------
公开后任何人都能消耗你的服务器算力。
建议尽快加：
  - 访问密码 / API Key
  - 请求频率限制
需要的话我可以下一步帮你加。
