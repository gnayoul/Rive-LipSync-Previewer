========================================
方案 A：免费公网上线
========================================

目标：
  别人用 Chrome / Edge / Safari 打开你的链接就能用
  文本生成语音由服务端完成（不要求用户装特定浏览器）

注意：
  - Render 免费档闲置会休眠，第一次打开可能要等 30～60 秒
  - 可用 UptimeRobot 每 5 分钟访问 /api/health 防休眠

----------------------------------------
部署
----------------------------------------
1. 代码推到 GitHub
2. Render → New Web Service → 连接仓库
3. Build: npm install
4. Start: node server.mjs
5. 获得 https://xxxx.onrender.com

打开该网址即可使用。
