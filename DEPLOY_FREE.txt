========================================
方案 A：免费公网上线（Edge 语音）
========================================

目标：
  别人用 Chrome / Edge / Safari 打开你的链接就能用
  语音走服务器上的 Edge TTS 代理（不要求用户装 Edge）
  Kokoro 仍可在你本机用，公网默认用 Edge

注意：
  - Render 免费档闲置会休眠，第一次打开可能要等 30～60 秒
  - 依赖微软 Edge TTS，长期可用性不保证，但够用且免费

----------------------------------------
第 1 步：注册 GitHub（如果还没有）
----------------------------------------
1. 打开 https://github.com/signup
2. 注册并登录

完成后回复：「GitHub 好了」

----------------------------------------
第 2 步：把项目上传到 GitHub（预告）
----------------------------------------
我会带你：
  - 创建仓库
  - 上传当前项目代码

----------------------------------------
第 3 步：用 Render 免费部署（预告）
----------------------------------------
1. 打开 https://render.com 注册（可用 GitHub 登录）
2. New → Web Service
3. 连接刚才的仓库
4. 自动获得一个 https://xxxx.onrender.com 地址

----------------------------------------
第 4 步：把链接发给别人
----------------------------------------
打开该网址即可使用。
语音引擎选「Edge 神经语音」即可。
