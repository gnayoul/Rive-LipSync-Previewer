Rive Rhubarb Lip Sync Previewer

推荐打开方式：
1. 双击 start.bat（或运行 node server.mjs）
2. 浏览器访问 http://localhost:3921
3. 页面需要联网（加载 Rive runtime 与嘴型分析 WASM）

----------------------------------------------------------------------
核心能力
----------------------------------------------------------------------

- 上传模式：.riv + 音频 → 自动生成嘴型 → 预览 / 下载 mouth.json
- 文本生成：输入台词 → 合成语音 → 自动分析嘴型 → 预览 / 导出
- 嘴型：浏览器内 Rhubarb（lip-sync-engine WASM）
- 语音：服务端 Edge TTS（对用户不披露引擎名称）
- 导出：speech.mp3、mouth.json

----------------------------------------------------------------------
使用步骤（上传模式）
----------------------------------------------------------------------

1. 素材来源 → 上传文件
2. 选择 .riv 与音频（嘴型会自动生成）
3. 可选：下载 mouth.json
4. 载入 Rive → 播放

----------------------------------------------------------------------
使用步骤（文本生成模式）
----------------------------------------------------------------------

1. 素材来源 → 文本生成
2. 选择 .riv，输入台词，选择声音与语速
3. 点击「生成语音与嘴型」
4. 载入 Rive → 播放
5. 可下载音频与 mouth.json

----------------------------------------------------------------------
本机依赖
----------------------------------------------------------------------

- Node.js 18+（运行 start.bat / server.mjs）
- 首次使用：在项目目录执行 npm install
