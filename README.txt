Rive Rhubarb Lip Sync Previewer

推荐打开方式：
1. 双击 start.bat（或运行 node server.mjs）
2. 浏览器访问 http://localhost:3921
3. 页面需要联网（加载 Rive runtime 与 Rhubarb WASM 引擎）

也可直接双击 index.html（Edge 免费语音可用；Azure / ElevenLabs 需语音代理）。

----------------------------------------------------------------------
核心能力
----------------------------------------------------------------------

- 上传模式：.riv + 音频 + Rhubarb mouth.json → 预览嘴型
- 文本生成：输入台词 → 合成语音 → 浏览器内 Rhubarb 分析嘴型 → 预览 / 导出
- 嘴型引擎：仅使用 Rhubarb（lip-sync-engine WASM，在浏览器中运行）
- 语音引擎：Edge 免费 / Azure API / ElevenLabs API
- 导出：生成后可下载 speech.mp3 与 mouth.json

----------------------------------------------------------------------
使用步骤（上传模式）
----------------------------------------------------------------------

1. 素材来源 → 上传文件
2. 选择 .riv、音频、mouth.json
3. 载入 Rive → 播放

----------------------------------------------------------------------
使用步骤（文本生成模式）
----------------------------------------------------------------------

1. 素材来源 → 文本生成
2. 选择 .riv，输入台词
3. 选择语音引擎与声音
   - Edge 神经语音：免费，建议用 Microsoft Edge 浏览器
   - Azure / ElevenLabs：需填写 API Key，并启动语音代理（start.bat）
4. 等待页面显示「Rhubarb 引擎已就绪」
5. 点击「生成语音与嘴型 JSON」
6. 载入 Rive → 播放
7. 可点击「下载音频」「下载 mouth.json」

----------------------------------------------------------------------
语音代理（仅 Azure / ElevenLabs 需要）
----------------------------------------------------------------------

若 PowerShell 中 npm 报错，请用以下任一方式启动：

  双击 start.bat
  node server.mjs
  npm.cmd start

默认地址：http://localhost:3921

----------------------------------------------------------------------
API Key 配置
----------------------------------------------------------------------

Azure Speech：在 Azure 门户创建语音服务，填写 Key 与 Region（如 eastasia）
ElevenLabs：在 elevenlabs.io 获取 API Key，可点击「加载 ElevenLabs 声音列表」

Key 仅保存在本机浏览器 localStorage，不会上传到第三方服务器（除对应 TTS 服务商）。

----------------------------------------------------------------------
Rive 嘴型映射
----------------------------------------------------------------------

mouth Number 值：X=0, A=1, B=2, C=3, D=4, E=5, F=6, G=7, H=8

----------------------------------------------------------------------
常见问题
----------------------------------------------------------------------

- Rhubarb 引擎加载失败：检查网络，确保能访问 cdn.jsdelivr.net
- Edge 语音失败：改用 Microsoft Edge 浏览器，或切换 Azure
- Azure / ElevenLabs 失败：先启动 start.bat，并用 http://localhost:3921 打开页面
- 嘴型与声音错位：调整「嘴型时间偏移 秒」（慢 -0.05，早 +0.05）
- npm 报脚本策略错误：用 start.bat 或 node server.mjs，不要直接用 npm start

----------------------------------------------------------------------
技术说明
----------------------------------------------------------------------

嘴型分析使用开源项目 lip-sync-engine（Rhubarb Lip Sync 的 WASM 移植）：
https://github.com/biolimbo/lip-sync-engine

首次加载约需下载 2MB+ WASM 与模型文件，之后浏览器会缓存。
